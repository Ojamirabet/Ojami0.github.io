kerbrute RECON
Kerberos User Enumeration

Enumera usuarios de dominio válidos a través del protocolo Kerberos sin autenticación previa.

impacket EXPLOITATION
AS-REP Roasting & DCSync

Suite de scripts Python para interactuar con protocolos de red Windows. GetNPUsers.py y secretsdump.py son clave en ataques AD.

Kerberos SMB DCSync NTDS.dit
hashcat CRACKING
Offline Hash Cracking

Herramienta de cracking de contraseñas basada en GPU. Imprescindible para crackear hashes Kerberos AS-REP (modo 18200).

GPU Wordlist AS-REP NTLM
smbclient ENUMERATION
SMB Share Enumeration

Cliente SMB de línea de comandos para listar y acceder a recursos compartidos de red en sistemas Windows.

SMB File Access Lateral Movement
evil-winrm POST-EXPLOITATION

Shell remota para Windows Remote Management. Soporta Pass-the-Hash, upload/download de ficheros y evasión básica de AV.
